## Phase 1: 1 페르소나 압도 (8주) — Quality 우위

> **타겟**: "월 매출 10~100만원, 쿠팡 파트너스 N개월차" 1명만 본다.
> **목표**: 이 1명이 "다른 도구 못 쓰겠다"고 말하게 만듦.

### 1-A. 핵심 가치 명확화 (2주)
- **"5분에 영상 1개 + 추적 링크 자동"** — 이 한 문장으로 모든 UI 정렬
- 현재 STEP 3의 8개 옵션 패널 → **3개 프리셋(빠른/표준/A·B)으로 단순화**
- 고급 설정은 접기

### 1-B. 추적 링크 자동 부착 (3주) — **해자 #1 시작점**
- 모든 영상 생성 시 쿠팡 파트너스 deeplink + 자체 추적 ID 자동 삽입
- 사용자 대시보드: "이 영상 → X뷰 → Y클릭 → Z원" 표시
- **다른 도구는 영상만 만들고 끝. 우리는 매출까지 본다.**

### 1-C. Quality 차별점 (3주)
- **수익률 기반 자동 재생성**: "이 영상 24시간 내 클릭 0회 → AI가 Hook 자동 변경 후 재생성 알림"
- **카테고리별 검증된 템플릿**: 인터뷰에서 수집한 "잘 나간 영상" 분석 → 카테고리별 프리셋
- **반복 가능한 품질**: 평가 메트릭 추가 (LLM 호출 편차 측정)

### 측정 지표 (매주 추적)
| KPI | 목표 |
|---|---|
| 주간 영상 생성 수 / 유저 | ≥ 5개 |
| 영상→업로드 전환율 | ≥ 60% |
| WAU/MAU | ≥ 50% |
| NPS | ≥ 40 |

**Go/No-Go**: 이 지표 미달 시 **Phase 2 진입 금지**.

## Phase 2: 데이터 해자 (12주) — **Moat 본격 구축**

> 기능은 6개월 안에 카피된다. **데이터는 카피 못 함**.

### 2-A. 성과 피드백 루프 (4주)
```
[영상 생성] → [추적 링크] → [클릭/뷰/매출] → [DB 누적]
                                      ↓
                          [AI 추천 모델 학습]
                                      ↓
                  [다음 영상 생성 시 "이 카테고리+가격대=이 Hook"]
```

핵심 테이블:
```sql
video_performance(
  video_id, category, price_range, hook_type, cta_text,
  views_24h, ctr, revenue
)
```

1,000개 영상 누적 시 **"카테고리 X + 가격대 Y에서 가장 잘 되는 Hook은 Z (자신감 87%)"** 추천 가능.

### 2-B. "AI 추천 적용 vs 미적용" A/B 강제 (4주)
- 모든 유저 50:50 분기 → **추천 적용 시 매출 +X%** 라는 숫자가 우리 유일한 마케팅 자산
- 6개월 후: "우리 도구 쓰면 평균 매출 +47%" — **카피 불가능한 해자**

### 2-C. 채널 OAuth 락인 (4주)
- 유튜브/틱톡/인스타 계정 연결 → 자동 업로드 + 댓글/조회수 회수
- 이탈 비용 = "다른 도구에서 다시 인증 + 영상 업로드 히스토리 잃음"
- Mirr가 가진 "업로드 자동화" 격차 해소 + 락인 동시 달성

### 측정 지표
- 데이터 누적 영상 수 (목표: 10,000개)
- AI 추천 적용군 vs 미적용군 매출 차이 (통계적 유의성)
- 채널 연동률 (목표: 활성 유저의 70%)

## Phase 3: TAM 확장 — 글로벌 (16주)

> **TAM-SAM-SOM 재정의**:
> - **TAM**: 글로벌 affiliate ($17B, Awin 2024) + 글로벌 e-commerce 셀러 광고 영상
> - **SAM**: **영상 기반 affiliate** = TikTok Shop ($16B GMV 2024, 5x 성장) + Amazon Influencer + Shopee Affiliate
> - **SOM 1년차**: TikTok Shop US 셀러 1만명 × ARPU $30/월 = $300K MRR 가능권

### 3-A. 인접 시장 우선순위

| 시장 | 성장률 | 진입 난이도 | 우선순위 |
|---|---|---|---|
| **TikTok Shop US/SEA** | 🔥 5x YoY | 중 (API 공개) | **1순위** |
| Amazon Associates (글로벌) | 안정 | 낮음 | 2순위 |
| Shopify D2C 셀러 광고 | 성장 | 중 | 3순위 |
| Shopee Affiliate (SEA) | 빠름 | 높음 (지역 특화) | 4순위 |
| 일본 모바이루 어필리엣 | 작음 | 낮음 | 5순위 |

### 3-B. 코드 전제 조건 (4주)
- **app.py 4,300+ lines 단일 파일 → 모듈 분리**
  - `core/` (영상 파이프라인) / `integrations/{coupang,amazon,tiktokshop}/` / `ui/streamlit/`
- **JSON 파일 저장 → SQLite/Postgres 마이그레이션** (멀티 유저 + 데이터 분석 위해 필수)
- **i18n** (ko/en/ja)

### 3-C. 글로벌 진입 (12주)
- 영어 UI + Reddit/IndieHackers/TikTok 셀러 커뮤니티 베타
- TikTok Shop API 연동 (자동 deeplink + 셀러 대시보드 매출 회수)
- "쿠팡" 단어 제거, **"E-commerce Affiliate Video Automation"** 으로 포지셔닝
- 한국 시장: "쿠팡 특화 모드" 토글로 유지 (기존 유저 보호)

## Phase 4: 플랫폼화 (장기) — 진짜 해자 완성

### 4-A. API 제공
- 영상 자동화 API → **Mirr·CapCut·Invideo가 "쿠팡/TikTok Shop 모듈"로 우리 API 사용**
- 도구 회사 → **인프라 회사**로 진화 ("선반 위 도구 → 선반 자체")

### 4-B. 데이터 마켓플레이스
- "Hook 템플릿 마켓" — 잘 나간 영상의 Hook을 다른 유저가 구매 (수익 분배)
- 네트워크 효과: 유저 많을수록 템플릿 풀 커지고 추천 정확해짐 — **클래식 데이터 네트워크 효과 해자**

### 4-C. B2B 화이트레이블
- 한국 인플루언서 에이전시·MCN 대상 화이트레이블 (월 수백만원 계약)
- 쿠팡 본사 카테고리 매니저용 셀러 영상 자동 생성 도구 (B2B 라이선스)

## 🛡️ 해자 (Moat) 4중 방어

| 해자 | 카피 난이도 | 시점 |
|---|---|---|
| **1. 매출 데이터 피드백 루프** | 🔴 매우 높음 (1만 영상 누적 시간) | Phase 2 |
| **2. 채널 OAuth 락인** | 🟠 높음 (이탈 비용) | Phase 2 |
| **3. AI 추천 정확도** ("우리 쓰면 +47%") | 🟠 높음 (데이터 없으면 불가) | Phase 2 종료 시 |
