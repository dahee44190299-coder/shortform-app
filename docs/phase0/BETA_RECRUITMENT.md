# 📣 베타 모집 — 30명 대기열 만들기

> **목표**: 1~2주 안에 신청 30명 확보 → 그중 인터뷰 5명 + 베타 사용 30명
> **기준**: 쿠팡 파트너스 운영 3개월 이상 + 영상 콘텐츠 시도 경험 有

---

## 📍 모집 채널 우선순위

| 채널 | 예상 반응 | 진입 난이도 | 우선순위 |
|---|---|---|---|
| 쿠팡 파트너스 네이버 카페 | 🔥 높음 (타겟 100% 일치) | 쉬움 | **1순위** |
| 쿠팡 파트너스 디스코드/오픈채팅 | 🔥 높음 | 중 (커뮤니티 매너 필요) | **1순위** |
| 인스타 #쿠팡파트너스 #쿠팡파트너스후기 DM | 중 | 중 (스팸 인식 위험) | 2순위 |
| 디스이즈마이팀 / 인디해커즈 한국 | 중 | 쉬움 | 2순위 |
| 트위터/X "쿠팡 파트너스" 검색 → 리플 | 낮음 | 쉬움 | 3순위 |
| 유튜브 쿠팡 파트너스 채널 댓글 | 낮음 | 쉬움 (스팸 위험) | 3순위 |
| 페이스북 부업 그룹 | 낮음 | 쉬움 | 4순위 |

> **주의**: 1순위 카페는 **운영 정책 확인 필수**. 무단 광고는 강퇴. "운영자 사전 동의"받고 게시.

```
요즘 영상 만드는 게 너무 오래 걸려서 자동화 도구를 만들고 있습니다.

[보너스]
🎁 30명 안에 들어오시는 분 평생 무료 사용권
🎁 인터뷰 30분 응해주시는 분 5,000원 커피 쿠폰 (5명 한정)

신청은 아래 링크에서:
👉 [Google Form 링크]

질문 있으시면 댓글/DM 편하게 주세요. 정말 감사합니다.
```

> **카페 매너**: 본인 인증/등업 후 게시. 광고로 분류되지 않게 "베타 테스터 모집" 카테고리 활용.

## 📋 Google Form 신청 양식

### 페이지 1: 자격 확인
```
1. 쿠팡 파트너스 운영 기간은? (필수)
   ○ 3개월 미만  → [자동 종료 화면: "지금은 3개월 이상 분만 모집 중이에요"]
   ○ 3개월~1년
   ○ 1년~3년
   ○ 3년 이상

2. 영상 콘텐츠 시도해보신 적 있나요? (필수)
   ○ 정기적으로 만들고 있음
   ○ 시도해봤지만 지금은 안 함
   ○ 한 번 만들어본 적 있음
   ○ 만들어본 적 없음 → [자동 종료]

3. 본인의 활동 채널은? (복수)
   □ 인스타 / □ 틱톡 / □ 유튜브 쇼츠 / □ 블로그 / □ 카페 / □ 기타
```

### 페이지 2: 매칭 정보
```
4. 한 달에 영상 콘텐츠 몇 개 정도 만드세요?
   ○ 0~3개  ○ 4~10개  ○ 11~30개  ○ 30개 이상

5. 영상 1개 만드는 데 평균 시간은?
   ○ 30분 이내  ○ 30분~1시간  ○ 1~3시간  ○ 3시간 이상

6. 한 달 쿠팡 파트너스 매출 (정확하지 않아도 OK)
   ○ ~10만원  ○ 10~50만원  ○ 50~200만원  ○ 200~500만원  ○ 500만원 이상
   ○ 답하기 어렵습니다

7. 30분 무료 인터뷰 응해주실 수 있나요? (사례비 5,000원 커피 쿠폰)
   ○ 네  ○ 아니오
```

### 페이지 3: 연락처
```
8. 이메일 (필수)
9. 카카오톡 또는 디스코드 ID (선택)
10. 마지막으로 한 마디 (선택, 자유)
```

## 🎯 30명 선별 기준 (신청자 50+ 발생 시)

우선순위:
1. **운영 기간 1년 이상 + 매출 50만원 이상** (페르소나 정합도 최고)
2. 인터뷰 응답 가능 표시
3. 영상 시도 경험 풍부 (월 4개 이상)
4. 페인 명확 (Q5 자유응답에 구체적 불만)

5명 인터뷰 대상은 위 우선순위 Top 5에서 선정.

## 📅 모집 일정 (4주 Phase 0 중 1~2주차)

```
Day 1   : 카페 운영자에게 사전 동의 요청
Day 2-3 : Google Form 만들기 + 모집글 게시 (1순위 채널 2~3곳)
Day 4-7 : 신청 수집 + 디스코드/오픈채팅 추가 모집
Day 8   : 30명 컷오프 + 5명 인터뷰 일정 조율
Day 9-14: 인터뷰 진행 (5명, 각 30분)
Day 15  : 인터뷰 분석 → METRICS_TRACKING.md 합산
Day 16-22: 30명 베타 사용 안내 + 1주 사용
Day 23  : 사용 후 설문 발송 (결제 의향 측정)
Day 24-28: 데이터 정리 + Go/No-Go 회고
```
