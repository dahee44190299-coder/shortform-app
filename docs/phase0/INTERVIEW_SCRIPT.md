# 🎙️ 사용자 인터뷰 스크립트 (Phase 0)

> **목적**: 쿠팡 파트너스 운영자의 진짜 페인 포인트를 찾고, 우리 가설을 검증한다.
> **대상**: 쿠팡 파트너스 운영 3개월~3년차, 월 매출 10만원~500만원, 영상 콘텐츠 시도 경험 有
> **시간**: 30분 (인사 3 + 본질문 22 + 마무리 5)
> **방식**: Zoom/Google Meet, 녹화 동의 필수, 노트는 [INTERVIEW_TEMPLATE.md](INTERVIEW_TEMPLATE.md) 사용

---

## 🚦 인터뷰 전 (사전 동의)

```
안녕하세요. 시간 내주셔서 감사합니다.

오늘 인터뷰는 [숏폼 자동화 도구]를 만들기 전, 실제 사용자분들의
이야기를 들으려고 진행합니다. 정답이 없는 대화이고, 어떤 답도
저희에게 도움이 됩니다.

녹화는 분석 용도로만 사용하고 외부 공유는 하지 않습니다.
녹화 진행해도 괜찮으실까요?

진행하기 전에 한 가지: 오늘 인터뷰의 사례비로
[ 5,000원 커피 쿠폰 / 베타 무료 평생 사용권 ]을 드립니다.
```

**체크**: 녹화 동의 ✅, 사례비 안내 ✅

---

## 🌱 1. 워밍업 (3분) — 라포 형성

```
1) 자기소개 부탁드릴 수 있을까요? (직업, 부업으로 파트너스를 하시는지 등)
2) 파트너스 시작한 지 얼마나 되셨어요?
```

**노트 포커스**: 본업 vs 부업, 운영 기간, 운영 동기

---

## 💔 2. 페인 포인트 발굴 (10분) — 가장 중요

> **금지 사항**: 우리 도구를 절대 먼저 언급하지 말 것. "이런 거 있으면 좋겠어요?" 질문 금지 (가설 오염).

```
3) 한 달에 영상 콘텐츠를 몇 개 정도 만드세요?
4) 영상 1개 만드는 데 보통 시간이 얼마나 걸려요?
   → (구체적으로) 어디에 가장 시간이 많이 들어가요?
   → 스크립트 작성? 영상 소스 찾기? 편집? 자막? 음성?
5) 가장 짜증 나는 단계가 어디예요? 왜요?
6) 최근에 만든 영상 중 가장 망한 거 있으세요? 왜 망했다고 생각하세요?
7) 반대로, 가장 잘 된 영상은요? 그게 왜 잘 됐다고 보세요?
8) 한 달 매출이 어느 정도세요? (말씀 어려우시면 대략 범위만)
9) 영상 만드는데 들어가는 비용 (도구, 외주, 시간 환산)이 얼마예요?
```

**노트 포커스**:
- ⏱️ 영상 1개 제작 시간 (분)
- 📊 월 영상 수
- 💰 월 매출 / 도구 비용
- 😤 가장 큰 페인 (구체적 단계)

---

## 🔧 3. 현재 도구 사용 행태 (5분)

```
10) 지금 영상 만들 때 어떤 도구 쓰세요?
    (CapCut / Premiere / 모바일 앱 / AI 도구 등)
11) 각 도구에 월 얼마 쓰세요?
12) 도구 바꿔본 적 있어요? 왜 바꾸셨어요?
13) 지금 도구의 가장 큰 불만은?
```

**노트 포커스**: 현재 도구 스택, 결제 의향 베이스라인 (지금 얼마 쓰는지), 이탈 트리거

---

## 🎯 4. 가설 검증 (7분)

> 우리 가설을 **풀이형 질문**으로 검증. 절대 "이런 거 만들 건데 어때요?"로 묻지 말 것.

```
14) 만약 영상 1개 만드는 시간을 5분으로 줄여주는 도구가 있다면,
    월에 얼마까지 낼 의향이 있어요? (가격 anchoring 없이 자유 답변)

15) 영상별로 "이 영상은 X뷰 / Y클릭 / Z원 매출" 이 한 화면에서 보이면
    얼마나 유용할 것 같아요? (현재는 어떻게 추적하세요?)

16) 만든 영상 중 어떤 게 잘 될지 미리 알 수 있다면 가치가 있을까요?
    예측 정확도가 어느 정도면 신뢰하시겠어요?

17) 쿠팡 파트너스 외에 다른 affiliate 시도해본 적 있어요?
    (아마존 어소시에이트 / TikTok Shop / 쿠팡 SEA 등)
    왜 안 하시는지 / 왜 시도하셨는지?
```

**노트 포커스**:
- 💵 자발적 호가 (anchoring 없는 가격)
- 📈 추적/대시보드 가치 인식
- 🌍 인접 시장 확장 의향 (Phase 3 단서)

---

## 🚪 5. 마무리 (5분)

```
18) 만약 베타 도구를 1주일 무료로 써보실 수 있다면, 어떤 기능이 있어야
    "오 이거 결제하겠다" 생각이 드실까요?

19) 비슷한 다른 분들 소개해주실 수 있어요?
    (1명만 소개해주셔도 정말 큰 도움입니다)

20) 마지막으로, 저희가 안 물어봤지만 꼭 말씀하시고 싶은 게 있으세요?
```

**클로징**:
```
정말 감사합니다. 사례비는 [방법]으로 보내드릴게요.
베타 오픈하면 가장 먼저 연락드릴게요.
```

## ⚠️ 인터뷰어 금기

| ❌ 금지 | ✅ 대신 |
|---|---|
| "이런 기능 있으면 쓰시겠어요?" (Leading) | "지금은 어떻게 해결하세요?" |
| "저희가 만들 도구는요..." (가설 오염) | 우리 도구 끝까지 비공개 |
| "가격이 1만원이면 어떠세요?" (Anchoring) | "얼마까지 내실 의향이세요?" 자유 답변 |
| "음... 그렇군요" (해석 끼어들기) | 5초 침묵 → 더 말하게 함 |
| 끄덕이며 동의 표현 | 중립적 표정 + 받아 적기만 |

## 🎯 인터뷰 5명 합산 분석 기준

5명 인터뷰 후 [METRICS_TRACKING.md](METRICS_TRACKING.md)에 합산:

- 평균 영상 1개 제작 시간
- 평균 월 영상 수 / 월 매출
- 가장 자주 언급된 페인 포인트 Top 3
- 자발적 호가 분포 (median, range)
- 인접 시장 진출 의향 비율
